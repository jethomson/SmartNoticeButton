ESP Web Tools - SmartNoticeButton

Plug your ESP32 into your computer and click Install to program it directly from this page.




After you have completed programming the ESP32, return to the README for instructions on setting up your device.





If you have trouble using this page you can try downloading the firmware, and programming it to the ESP32 using Espressif's site.
  • Download merged_firmware.bin
  • Go to the ESP Tool site
  • Click Connect
  • Choose your serial port when the browser asks
  • Click Choose File and select merged_firmware.bin
  • Set Flash Address to 0
  • Click Program
  • If your ESP32 does not restart automatically:
    • Unplug the ESP32 and plug it back in
    •     OR
    • Click Start, choose your serial port when the browser asks, and click Reset
